# Overview - S.A.P.O

## 1. Descripción del Proyecto

S.A.P.O (Sistema de Alerta de Partículas y Oxígeno) es un sistema embebido basado en ESP32 diseñado para detectar presencia de humo y monitorear la calidad del aire en espacios cerrados.

El sistema activa alertas visuales y sonoras en caso de detectar condiciones críticas y permite monitoreo en tiempo real mediante una interfaz web local.

---

## 2. Objetivo del Proyecto

Diseñar e implementar un sistema embebido capaz de:

- Detectar condiciones de posible incendio.
- Emitir alertas físicas.
- Permitir desactivación física y remota.
- Almacenar histórico de datos durante 7 días.
- Mostrar gráficas en una interfaz web local.

---

## 3. Factibilidad

### Factibilidad Técnica
El proyecto es viable debido al uso de la ESP32, la cual cuenta con conectividad WiFi, ADC y memoria suficiente.

### Factibilidad Económica
Los componentes utilizados son de bajo costo y fácil acceso.

### Factibilidad Operativa
El sistema funciona en red local sin necesidad de internet.
